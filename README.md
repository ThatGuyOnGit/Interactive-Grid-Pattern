# 3x3-Interactive-Grid-Pattern-Capture

> this code snippet can be utilised with general left mouse button click button 
> this is based on tkinter module, tkinter events <Button-1> for LMB click has been used
> tkinter events <Button-1>, <B1-Motion>, <ButtonRelease-1> used 

Grid Layout:

0  1  2

3  4  5

6  7  8

Input Example:

![Single LMB click on 8](image.png) #single LMB click on 8
output---> `Pattern Captured: [8]`

![Line LMB drag on 0, 4, 8](image-1.png) #line LMB drag on 0, 4, 8
output---> `Pattern Captured: [0, 4, 8]`


Output description:
|
------------> `Pattern Captured: [0, 4, 8, 7, 5, 1]` 
|
------------> `Pattern Captured: [3]`
